# Angry Birds Game

A single-file HTML5 canvas game clone with procedural audio and enhanced gameplay.

## Features
- Slingshot aiming with trajectory preview (dotted arc)
- Power meter showing shot strength
- Procedural sound effects (Web Audio API, no external files needed)
- Mid-flight dash ability (tap/click while bird is flying)
- Multiple levels with increasing difficulty
- Particle effects for destruction

## How to Play
1. Open `AngryBirds.html` in a browser
2. Drag the slingshot back and release to launch birds at pigs
3. While a bird is flying, tap/click it to activate dash ability (once per bird)
4. Destroy all pigs to complete each level

## Tech
- Pure HTML5 Canvas + JavaScript — zero dependencies
